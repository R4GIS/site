O melhor de 50 posts!

Por William 05/08/2017

Nosso blog completou 50 posts e para não deixar a marca passar em branco, elaboramos um resumão do que de melhor aconteceu aqui nos últimos 5 meses.

Mas antes, gostaríamos de agradecer a todos que nos acompanham, deixam comentários, correções ou sugestões e interagem com a gente nas redes sociais ou nos corredores da vida. Para nós, é uma satisfação muito grande falar de R e Estatística. Nós aprendemos e nos divertimos muito fazendo esse blog, e é muito bom poder compartilhar isso com vocês. s2

Agora, o resumão!

Análises

A seção de mais sucesso do nosso blog foi a de análises. Nós usamos diversas bases de dados divertidas e interessantes para mostrar os recursos do R de webscraping, visualização e modelagem.

  • Utilizando a API do YouTube, coletamos dados de visualizações do Porta dos Fundos e discutimos se o canal estava em decadência. E revisitamos assunto algum tempo depois, abordando os dados de uma outra maneira.

  • Usamos dados do Spotify para testar hipóteses sobre a organização das bandas no Lollapalooza.

  • Brincamos com imagens, aprendendo como estrair uma paleta de cores de uma figura e utilizamos isso para analisar as cores dos filmes da Marvel e da DC.

  • Com dados públicos, analisamos a distribuição das multas em São Paulo e construímos um mapa da criminalidade da cidade.

  • Utilizamos a API Microsoft Emotions para decifrar as emoções do Aécio Neves e a API do GoogleMaps para descobrir o quanto estamos longe de uma hamburgueria.

Conceitos

Ensinar é uma paixão dentro da Curso-R, e tentamos passar adiante não apenas conhecimento técnico, mas também discutir conceitos de temas ligados tanto ao R quanto à Estatística. Veja a lista do que discutimos nesses durante esses 50 posts:

  • O manifesto tidy
  • O que é um gráfico estatístico
  • Como fazer requisições web seguras
  • O que são os environments dentro do R
  • Componentes principais

  • A construção de autoencoders
  • A relação entre regressão logística e deep learning

Tutoriais

De longe a seção com mais posts do blog, os tutoriais são uma forma rápida de apresentar pacotes, estratégias de análise e como resolver problemas específicos dentro do R. Segue uma lista dos pacotes que apareceram por aqui:

  • ggalt, uma extensão do ggplot2 que constrói intervalos de confiança para as curvas de Kaplan-Meier.

  • leaflet, para construir mapas interativos no R.

  • progress, para construir barras de progresso.

  • glue, um ótimo pacote para colar textos.

  • VennDiagram, para construir diagramas de Venn.

  • magick, para manipular imagens.

  • skimr, para fazer estatísticas básicas.

  • forcats, para manipular fatores.

  • tidywoe, para a construção de WoE (weight of evidence).

  • greta, para modelagem estatística usando MCMC.

E não para por aí, também fizemos tutoriais mostrando como modelar a variância da normal, como mexer com paralelização no R, como tratar com erros de uma maneira tidy, como extrair dados de um arquivo pdf e como incluir o github na sua rotina de programação.
